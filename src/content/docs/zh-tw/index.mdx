## 先決條件

開始之前，您應該熟悉：

- **Python 3.10+** — 所有程式碼範例使用現代 Python
- **API 基礎** — 理解 REST API 和非同步程式設計
- **Claude API** — 從 [console.anthropic.com](https://console.anthropic.com/) 取得您的 API 金鑰

不需要 AI/ML 背景！這些教學從第一原理開始教授代理模式。

## 學習路徑

1. **[案例研究概覽](/ai-agent-study/zh-tw/case-study/)** — 理解問題領域並比較模式（15 分鐘）
2. **[ReAct 模式](/ai-agent-study/zh-tw/react/01-overview/)** — 學習簡單模式（從這裡開始！）

3. **[計劃-執行-驗證](/ai-agent-study/zh-tw/plan-execute-verify/01-overview/)** — 擴展至生產級
   - 含圖表的概覽 → Claude SDK → 模型無關 → LangChain
